# Tetris MVP - 구현 계획

## 개요
브라우저에서 바로 실행 가능한 테트리스 MVP 게임.
빌드 도구나 외부 의존성 없이 HTML/CSS/JS 분리 파일로 구성합니다.

## 파일 구조
```
tetris/
├── plan.md      — 이 파일
├── index.html   — 랜딩 페이지 (게임 소개, game.html 링크)
├── game.html    — 게임 페이지 (캔버스, 사이드 패널, game.js 로드)
├── style.css    — 레이아웃, 색상, 폰트 등 모든 스타일
└── game.js      — 게임 로직 전체
```

## 기능 목록 (MVP)
1. **7종 테트로미노**: I, O, T, S, Z, J, L (각기 다른 색상)
2. **블록 이동**: ← → (좌우), ↓ (빠른 낙하), ↑ (회전), Space (즉시 낙하)
3. **자동 낙하**: 레벨에 따라 속도 증가
4. **줄 제거**: 가로 줄이 가득 차면 삭제 + 점수 추가
5. **점수 표시**: 제거한 줄 수 × 100 (멀티라인 보너스)
6. **게임 오버**: 블록이 상단에 쌓이면 종료 + 재시작 버튼
7. **다음 블록 미리보기**: 사이드 패널에 표시

## 파일별 역할

### index.html
- 랜딩 페이지: 게임 소개, 조작법, 점수 계산표
- `game.html`로 이동하는 플레이 버튼
- `<link>` → style.css 로드

### game.html
- `<canvas id="board">` (게임 보드)
- 점수·레벨·다음 블록 표시용 사이드 패널 HTML
- `<link>` → style.css, `<script>` → game.js 로드

### style.css
- 전체 레이아웃 (flex, 중앙 정렬)
- 어두운 배경(#1a1a2e), 네온 느낌 블록 색상
- 사이드 패널 스타일, 게임오버 오버레이

### game.js
- 7종 테트로미노 정의 및 색상
- 게임 상태 관리 (board, currentPiece, score, level)
- 게임 루프 (`requestAnimationFrame` + 타임스탬프)
- 충돌 감지, 줄 제거, 점수 계산
- 키보드 이벤트 처리 (Arrow keys + Space)
- 다음 블록 미리보기 렌더링

## 기술 스택
- 순수 HTML5 + CSS3 + Vanilla JavaScript
- `<canvas>` API로 게임 보드 렌더링
- 외부 라이브러리 없음

## 검증 방법
1. 브라우저에서 `index.html` 파일을 직접 열어 실행
2. 블록이 자동으로 낙하하는지 확인
3. 키보드 좌/우/아래/회전/즉시낙하 동작 확인
4. 줄이 가득 차면 삭제되고 점수가 올라가는지 확인
5. 게임 오버 시 재시작 가능한지 확인
