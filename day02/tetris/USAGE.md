# Tetris 풀스택 애플리케이션 - 사용 가이드

## 시작하기

### 1. 서버 실행

터미널에서 tetris 디렉토리로 이동한 후:

```bash
# 백엔드 서버 실행 (백그라운드)
python3 backend.py > backend.log 2>&1 &

# 프론트엔드 서버 실행
python3 -m http.server 8080
```

### 2. 브라우저에서 접속

웹 브라우저를 열고 `http://localhost:8080`으로 이동합니다.

### 3. 회원가입

1. 홈 화면에서 "게임 시작" 버튼을 클릭
2. 로그인 페이지로 이동
3. "회원가입" 탭으로 전환
4. 이메일과 비밀번호 입력 (최소 6자)
5. "회원가입" 버튼 클릭

### 4. 게임 플레이

회원가입이 완료되면 자동으로 게임 페이지로 이동합니다.

#### 조작법
- `← →` : 블록 좌우 이동
- `↑` : 블록 회전
- `↓` : 빠른 낙하 (소프트 드롭)
- `Space` : 즉시 낙하 (하드 드롭)
- `P` : 일시정지/재개

#### 게임 화면
- **SCORE**: 현재 점수
- **LEVEL**: 현재 레벨 (속도)
- **LINES**: 클리어한 라인 수
- **TOP SCORE**: 전체 사용자 중 최고 점수와 플레이어
- **NEXT**: 다음에 나올 블록

### 5. 점수 시스템

- 1줄: 100 × 레벨
- 2줄: 300 × 레벨 (Double)
- 3줄: 500 × 레벨 (Triple)
- 4줄: 800 × 레벨 (Tetris!)

### 6. 레벨업

- 30초마다 자동으로 레벨 상승
- 10줄을 클리어할 때마다 레벨 상승
- 레벨이 오르면 블록 낙하 속도 증가

### 7. 게임 종료

- 블록이 맨 위에 쌓여 새 블록이 나올 수 없을 때 게임 종료
- 게임 기록이 자동으로 서버에 저장됨
- 최고 점수 갱신 시 TOP SCORE에 즉시 반영

### 8. 리더보드 확인

- 홈 화면에서 "🏆 리더보드" 버튼 클릭
- 전체 사용자의 상위 20개 기록 확인 가능
- 각 기록의 점수, 레벨, 라인 수, 플레이 날짜 표시

### 9. 로그아웃

- 게임 화면 상단의 "로그아웃" 버튼 클릭
- 로그인 페이지로 이동

## 서버 종료

```bash
# 백엔드 서버 종료
pkill -f "python3 backend.py"

# 프론트엔드 서버 종료 (Ctrl+C)
```

## 문제 해결

### 포트가 이미 사용 중인 경우

```bash
# 8001 포트 사용 중인 프로세스 확인
lsof -i :8001

# 해당 프로세스 종료
kill -9 <PID>

# 또는 다른 포트 사용 (backend.py와 프론트엔드 파일들의 API_BASE 수정 필요)
```

### 백엔드 서버 연결 실패

1. 백엔드 서버가 실행 중인지 확인: `ps aux | grep backend.py`
2. 로그 확인: `cat backend.log`
3. 서버 재시작: `pkill -f backend.py && python3 backend.py &`

### 데이터베이스 초기화

```bash
# 기존 데이터베이스 삭제 (주의: 모든 데이터 손실)
rm tetris.db

# 서버 재시작 시 자동으로 새 DB 생성
python3 backend.py
```

## API 문서

백엔드 서버 실행 후 `http://localhost:8001/docs`에서 인터랙티브 API 문서를 확인할 수 있습니다.

## 기술 스택

- **Backend**: FastAPI (Python)
- **Database**: SQLite
- **Frontend**: Vanilla JavaScript, HTML5 Canvas
- **Authentication**: Bearer Token
